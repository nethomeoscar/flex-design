Hola, estamos aquí para ayudarte
  • Herramienta pluma: se activa con el botón correspondiente o con la tecla P. Sirve para realizar trazados.
  • Herramienta selección: se activa con el botón o con la tecla V. Permite seleccionar nodos.
  • Desplazar lienzo: presiona la barra espaciadora y arrastra.
  • Zoom: usa la rueda del mouse.
  • Propiedades del trazado: en el panel lateral puedes elegir entre línea recta o curva, continua, punteada o discontinua.
  • Nodos y manipuladores: en curvas se usan manipuladores, que pueden ser simétricos o independientes.
  • Capas: pestaña para observar, seleccionar o eliminar trazos en el lienzo.
  • Imágenes: pestaña para insertar imágenes de referencia (solo una a la vez).
Acerca de NeónFlex Design

Este programa es propiedad de CarriotBrain Development, y está diseñado para la creación y edición de trazados vectoriales de manera intuitiva.

Combina herramientas rápidas de teclado con paneles visuales para que el usuario pueda trabajar de forma ágil y organizada.

Su objetivo es ofrecer un entorno accesible para estudiantes, diseñadores y creativos que necesiten producir gráficos precisos y personalizables.

CarriotBrain Development es una división de CarriotBrain Technologies.
